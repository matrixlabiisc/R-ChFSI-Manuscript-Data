# R-ChFSI: Residual-based Chebyshev Filtered Subspace Iteration

Companion code for the paper:

> **Residual-based Chebyshev filtered subspace iteration for Hermitian eigenvalue
> problems tolerant to inexact matrix-vector products**
> *N. Kodali, K. Ramakrishnan, P. Motamarri — Computer Physics Communications*

This is a self-contained C++17 implementation of Algorithms 1–3 from the paper,
applied to dense symmetric (generalised) eigenvalue problems with controlled
noise.  It produces the convergence data behind Figures 1–3 (Section 4.1).

---

## Paper-to-Code Map

### Algorithms

| Paper | Name | Code | Description |
|:------|:-----|:-----|:------------|
| **Algorithm 1** | ChFSI outer loop | `solve()` in [src/eigensolver.cpp](src/eigensolver.cpp) | Filter → Rayleigh-Ritz → check convergence. Iterates until max residual < τ. |
| **Algorithm 2** | Standard Chebyshev filter | `chfsi_filter()` in [src/chebyshev_filter.cpp](src/chebyshev_filter.cpp) | Three-term recurrence: Y = C_p(D⁻¹A) X. Eq. (2). |
| **Algorithm 3** | R-ChFSI (residual-based) filter | `rchfsi_filter()` in [src/chebyshev_filter.cpp](src/chebyshev_filter.cpp) | Recurrence on residuals R = AX − BXΛ. Eqs. (8)–(10). Robust to inexact D⁻¹. |

### Key Equations

| Equation | What it represents | Where verified in code |
|:---------|:-------------------|:-----------------------|
| **Eq. (1)** Au = λBu | Generalised eigenvalue problem | Solved by `solve()` in [src/eigensolver.cpp](src/eigensolver.cpp) |
| **Eq. (2)** | ChFSI three-term recurrence | Inner loop of `chfsi_filter()` in [src/chebyshev_filter.cpp](src/chebyshev_filter.cpp) |
| **Eq. (5)** convergence condition | \|C_p(λ_n)\| − \|C_p(λ_{n+1})\| > \|Δ̂_p\|(sec θ + csc θ) | Evaluated per-iteration in `solve()` when `track_eq5 = true`; stored in `IterRecord.eq5_lhs/eq5_rhs` |
| **Eq. (8)** | R-ChFSI residual recurrence (Z vectors) | Inner loop of `rchfsi_filter()` (the `RX`, `RY`, `Rnext` variables) |
| **Eq. (9)** | Eigenvalue polynomial recurrence (Λ_k) | `LambdaX`, `LambdaY`, `LambdaNext` in `rchfsi_filter()` |
| **Eq. (10)** | R-ChFSI with inexact products | Same as eq. (8) implementation — inexactness enters through `Dinv` and optional `A_filter` |

## How solve() Maps to Algorithm 1

The function `solve()` in [src/eigensolver.cpp](src/eigensolver.cpp) follows Algorithm 1 directly:

1. **Step 0 — Initial guess:** Random X^(0) of size m × (n + n_extra), then an initial
   Rayleigh-Ritz to get Λ^(0).

2. **Step 1 — Chebyshev filter:** Calls `chfsi_filter()` (Alg. 2) or `rchfsi_filter()` (Alg. 3)
   depending on `SolverConfig.method`.

3. **Step 2 — Rayleigh-Ritz:** Projects the filtered vectors Y_p onto the small
   (n_sub × n_sub) generalised eigenvalue problem via `rayleigh_ritz()` (static function),
   using LAPACK `dsygv`.

4. **Step 3 — Subspace rotation:** X^(i+1) = Y_p · E (the Ritz vectors), done inside
   `rayleigh_ritz()`.

5. **Convergence check:** max_j ‖Ax_j − λ_jBx_j‖ < τ.

6. **Diagnostics:** Each iteration records residual norms, subspace angles, and the eq. (5)
   condition into `IterRecord`, matching the quantities plotted in Figures 1–3.

## Test Problem Construction (Section 4.1.1)

`build_test_problem()` and `build_standard_problem()` in [src/test_problems.cpp](src/test_problems.cpp) create
the controlled-noise experiments from Section 4.1.1:

- **Standard (B = I):** A = Q diag(eigs) Q^T with perturbed matvec A_filter = A + ε·E, ‖E‖₂ = 1.
- **Generalised (B ≠ I):** A, B built with prescribed spectra; D⁻¹ = B⁻¹ + ζ·E, ‖E‖₂ = 1.
- Exact eigenpairs computed by LAPACK `dsygv` for ground-truth comparisons.

## Prerequisites

- C++17 compiler (GCC ≥ 9, Clang ≥ 10)
- CMake ≥ 3.16
- BLAS + LAPACK (OpenBLAS, MKL, FlexiBLAS, etc.)
- gnuplot (for figure generation only)

## Build and Run

```bash
mkdir -p build && cd build
cmake ..
make -j
./plot_convergence ../data          # regenerate .dat files
cd ../data && gnuplot make_paper_plots.gp   # regenerate PDF figures
```
